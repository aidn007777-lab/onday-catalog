# ONDAY Catalog

Технический каркас Next.js-приложения для ONDAY Catalog. На этом этапе реализован только демонстрационный интерфейс на локальных тестовых данных.

## Что уже есть

- Next.js App Router, TypeScript, Tailwind CSS и ESLint.
- Темный интерфейс ONDAY Catalog с адаптацией под компьютер, планшет и смартфон.
- Локальные переводы RU/KZ.
- Демо-таблица, демо-карточки и правая карточка товара.
- Тестовые публичные склады: `Тараз — склад 1` и `Алматы — склад 1`.

## Что сознательно не подключено

- Supabase, SQL-миграции и реальные таблицы.
- Настоящая авторизация.
- Импорт WhatsApp-прайсов.
- Telegram-уведомления.
- Реальные товары, закупочные цены и настоящие поставщики.

## Запуск

```bash
npm install
npm run dev
```

После запуска приложение будет доступно по адресу:

```text
http://localhost:3000
```

## Проверки

```bash
npm run typecheck
npm run lint
npm run build
```
